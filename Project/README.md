# Installations
- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt

# Run teacher code
- source venv/bin/activate
- python3 teacher/mainServer.py

# Run student code
- source venv/bin/activate
- python3 student mainClient.py
- key in **name** and **id** 
- give sudo permission for port scanning
- press enter to start
